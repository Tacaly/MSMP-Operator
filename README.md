# MSMP-Operator
Minecraft server website with a management backend using the new MSMP



### Minecraft Server Setup (MSMP)
Put this in the "server.properties" file.

'''
management-server-enabled=true
management-server-host=127.0.0.1
management-server-port=25585
management-server-secret="YOUR_40_CHAR_SUPER_SECRET_KEY"
management-server-tls-enabled=false
'''
